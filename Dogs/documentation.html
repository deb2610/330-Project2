A Deeper Look Into Fetch!

Fetch! is a lightweight web app that utilized The PetFinder API and Mapbox to match you with a perfect adoptable cat or dog. The initial idea for the app was to make a "Tinder for Dogs." However after some research we found that Petfinder's API was able to offer certain information about pets, making the dating profile setup more challenging to achieve


This application was done as a final project for Rich Media, the second Web Design and Development class in our curriculum.The following is a breakdown of how we satisfied the requirements

Criteria Description
A. Functionality
1. TWO web services are used Mapbox and Petfinder API
2. Cloud storage is used Firebase stores the data for user wishlists
3. Last term searched stored in browser Search parameters are stored
4. Has required Controls (3) Three parameters control the search
5. Runs without errors
B. Design & Interaction
1. Visual design is pleasing
2. Widgets are well labeled and follow interface conventions
3. Users should be able to figure out how to use the app with minimal instruction
4. User errors must be handled gracefully
5. The state the application is in is obvious We have a loading state show if the results are pending
6. The app should look good on a range of displays. Optimized for 1920 x 1080 to mobile
C. HTML/CSS/Media
1. Valid HTML
2. Valid CSS
3. Images properly optimized
D. Code
1. MVVM Framework used We use Vue.js
2. Ajax used
3. ES6 Custom Class Used for the Map
E. Impact
1. Does the app work as intended and do something useful? The app works with minimal errors and hooks you up with cute dogs and cats
2. Does the app functionality and programming go beyond what we did in class?
3. Is this project "portfolio quality" that you would not hesitate to show a potential employer?
4. Above and Beyond Full Mobile Optimization
